# Demo Script

## Short Demo Flow

Open the app homepage and say:

`Which region has the highest revenue?`

Explain:

- the system captures speech in the browser
- it translates the request into a safe SQL plan
- it queries the database
- it responds with a spoken business insight

Then ask:

`Show the monthly profit trend for Alpha`

Explain:

- the app is now handling a trend analysis request
- it returns both a summary and supporting rows
- the user can review the generated SQL for transparency

Then ask:

`What about North?`

Explain:

- the assistant remembers the earlier context
- it keeps the same metric and analysis mode where appropriate
- this makes follow-up questions feel conversational

Then ask:

`What are the biggest risk hotspots?`

Explain:

- the system can move beyond raw query responses
- it highlights operational risk patterns and anomalies
- this is useful for decision support, not just reporting

## Short Voiceover Script

This project is a voice-based AI SQL insight agent. A user speaks a business question, the system converts it into a secure query plan, analyzes internal data, and answers with a concise spoken insight. It also supports follow-up questions with context memory, so the experience feels conversational rather than form-based. In addition to direct answers, the agent can highlight trends, low-performing areas, and risk hotspots through a lightweight browser interface.

## Suggested Demo Length

- 60 to 90 seconds for freelancing platforms
- 2 to 3 minutes for direct client demos
