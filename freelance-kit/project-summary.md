# Project Summary

## One-Line Version

Voice-based AI analytics agent that turns spoken business questions into secure SQL queries and responds with spoken insights, trends, and risk analysis.

## Portfolio Version

Built a voice-enabled AI data assistant that accepts spoken questions, converts them into secure SQL queries, analyzes structured business data, and returns clear spoken and visual insights through a lightweight browser interface.

Key capabilities:

- Speech-to-text input for hands-free querying
- Text-to-speech answers for natural conversational output
- Natural language to SQL conversion with safe controls
- Follow-up context memory for multi-turn conversations
- Trend, anomaly, and risk-oriented analysis
- Transparent SQL and result display
- Ready for SQLite and adaptable to MySQL or PostgreSQL

## Gig Description Version

I build voice-based AI data assistants that can understand spoken business questions, translate them into secure SQL queries, analyze internal databases, and explain the results in plain language. The system supports follow-up questions, voice replies, and insight-driven analytics such as trend detection, anomaly review, and operational risk identification.

Ideal use cases:

- sales performance analysis
- operational monitoring
- customer support analytics
- executive reporting assistants
- internal business intelligence tools
