# Scope And Milestones

## Core Scope

The deliverable includes:

- Voice input using speech-to-text
- Voice output using text-to-speech
- Natural language to SQL/query conversion
- Context memory for follow-up questions
- Real-time analysis focused on insights, not just raw rows
- Secure database connection layer
- Browser-based interface
- Source code
- Deployment instructions
- User guide

## Suggested Milestones

### Milestone 1: Prototype

- Confirm business questions and schema
- Build base UI
- Connect to sample or client database
- Implement first-pass natural language to query flow

### Milestone 2: Intelligence Layer

- Add context-aware follow-up handling
- Add trend, anomaly, and risk analysis logic
- Improve answer formatting and voice output
- Add secure query validation rules

### Milestone 3: Production Readiness

- Integrate live database credentials securely
- Add logging, testing, and deployment setup
- Final UI refinements
- Deliver documentation and handoff package

## Timeline Example

- Prototype: 2 to 4 days
- Intelligence layer: 3 to 5 days
- Production polish and handoff: 2 to 3 days

Total estimated delivery window:

- 1 to 2 weeks depending on database complexity and deployment requirements

## Out Of Scope Unless Requested

- complex user authentication
- multi-database tenant architecture
- advanced charting dashboards
- custom mobile apps
- on-prem enterprise deployment approvals
